RESTful API

Full HTTP API for hash cracking, team management, and wordlist operations

POST /api/crack_hash Submit hash for cracking
GET /api/identify_hash Identify hash type
POST /api/create_team Create team room

Example Usage

# Crack a hash
curl -X POST ${CONFIG.API_BASE_URL}/api/crack_hash \
  -H "Content-Type: application/json" \
  -d '{
    "hash": "5f4dcc3b5aa765d61d8327deb882cf99",
    "wordlists": ["rockyou"],
    "mode": "solo"
  }'

# Response
{
  "success": true,
  "plaintext": "password",
  "hash_type": "MD5",
  "time_taken": 0.23,
  "engine_used": "john"
}
